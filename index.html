Data Provenance & Methodology

Our Sources

VoteWatch USA aggregates data from authoritative primary sources. No editorializing — raw vote records only.

📡

Congress.gov API

Primary source for real-time roll-call votes, member profiles, and bill tracking. Free API with registration. Updated within hours of each vote.


api.congress.gov →
🏛️

Congress.gov

Official US Library of Congress database. Used to verify bill text, official vote tallies, and legislative history for all federal legislation.


congress.gov/roll-call-votes →
📊

GovTrack.us

Independent tracker providing voting scorecards, ideological analysis, and historical vote context for all members of Congress since 1789.


govtrack.us/congress/votes →
🗺️

State Legislature Portals

Each state links directly to its official legislature portal (e.g., legis.ga.gov, leg.colorado.gov). State voting data is not centralized — direct links are provided per state.

🔄

Update Frequency

Federal votes: updated within hours via Congress.gov API (official Library of Congress source). State votes: sourced directly from each state's portal. Page refresh triggers new API calls.

⚖️

Nonpartisan Commitment

VoteWatch USA displays raw voting records without editorial framing. Party affiliations and vote labels (Yea/Nay) use official congressional records only.
